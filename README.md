## About

PC Privacy Tools is an independent project built by a privacy and security practitioner with **9+ years of experience in online privacy, security tooling, and threat modelling**.

The goal is simple:

- build **practical privacy tools**
- publish **clear technical guides**
- avoid **ads, tracking, and affiliate bias**

Most tools are designed to run **entirely in the browser**, meaning:

- no accounts required
- no server-side logging
- no data leaves the device
- fully inspectable source

## Philosophy

Privacy advice online usually falls into two extremes:

- vague reassurance that changes nothing  
- fear-based marketing designed to sell products

PC Privacy Tools focuses on **practical improvements that actually matter**, explained clearly and without hype.

Sometimes the honest answer is that something **doesn't meaningfully improve privacy** — and that is stated directly.
